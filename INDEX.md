## üéØ Quick Reference

### Essential Commands
```bash
npm install          # Install dependencies
npm run dev          # Start dev server
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run linter
```

### Essential Files to Customize
```
src/tokens/design-tokens.ts    # Colors, sizes, spacing
src/types/index.ts             # Add new types
src/utils/geometry.ts          # Modify calculations
```

### Essential Concepts
- **Design Tokens:** Single source of truth for visual properties
- **Geometry Engine:** Pure functions for shape calculations
- **Constraints:** Intentional limitations for consistency
- **Auto-Layout:** Content-driven sizing (pill component)
